Utilizar:
    nodejs -> versión: v14.15.1
    npm -> versión: 6.14.8

Instructions:

    $ git clone https://github.com/marlon2794/roomvr--aframe-wbrtc.git
    $ cd roomvr--aframe-wbrtc
    $ nmp install

Run server:
    
    $ npm run devStart


Run peer js with ssl certificate
    
    $ peerjs --port 3001 --sslkey key.pem --sslcert certificate.pem
